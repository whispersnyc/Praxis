Praxis: Information Management

The purpose of this system is to manage all different kinds of content on your phone and PCs, with a hyperfixation on efficiency/minimalism
Uses versatile industry standards like Markdown and, tech-wise, mostly just basic file/text operations to maximize compatibility and efficiency

On Android: Syncthing (Settings > Behavior > Autostart), Automate (certain Flows like Share) and Edge Shortcuts (automatic) must run at startup

Quickpooling

Quickpooling is capturing pics/vids/audio/ideas/links/files with minimal friction
Everything pools into only one p2p-synced folder, no categorization

  • Should prevent having to routinely cycle thru all the diff services/destination when its time to categorize or, even worse, search for something specific

I used Automate to "program" these systems on Android with Edge Shortcuts

Text

A popup with multi-line input box, cancel and confirm buttons
If cancel pressed, close popup
If confirm, append '\n<!--- YYYY-mm-dd HH:mm:ss -->\n{input}' to QUICKPOOL.md

  • ex. {newline}<!--- 2025-03-05 08:39:01 -->{newline}{input}
  • format {input} to escape (aka. insert \ symbols before) hashtags

In Automate, its easy to implement everything except the input formatting on Android
On Windows, I kitbashed a Tkinter Python popup script together with Claude

Clipboard/Share

The clipboard/share function can hold many formats of data: image, text
